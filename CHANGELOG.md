# 美诗儿 MES 操作指导书 — 变更日志

## v1.0 — 2026-07-03

### 新增
- 首个完整版本，包含 7 个操作模块
- 固定侧边栏 + 全宽响应式布局
- 横向 SVG 流程图（从左到右）
- 图片点击放大（lightbox）
- 滚动时自动高亮当前章节（ScrollSpy）

### 模块清单
- 注塑车间操作手册（生产部）
- 组装车间操作手册（生产部）
- 设备维保操作手册（设备部）
- 模具维保操作手册（模具部）
- 质量检验操作手册（质量部）
- IIoT 设备看板操作手册（生产部/设备部）
- 系统基础配置操作手册（信息部）

### 技术特性
- Python 生成器（`generate.py`），内容与样式分离
- 纯静态 HTML/CSS/JS，无第三方依赖
- 从 Word .doc 提取 PNG 图片（olefile）
- 自适应移动端布局

### 变更记录
- 去除 APS 排产模块（客户需求调整）
- 流程图改为横向排列，适配宽屏显示
- 流程图点击可放大查看细节
- 侧边栏固定显示，支持模块间快速切换
